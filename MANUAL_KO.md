# Stock Decision Helper 상세 매뉴얼

## 1. 프로그램 개요

이 프로그램은 한국/미국 주식을 함께 보면서 아래 3가지를 한 번에 처리하려는 목적의 개인용 투자 보조 앱입니다.

- 내 보유 종목 관리
- 오늘 볼 만한 추천 후보 확인
- 시간이 지나면서 추천 성과를 추적하고 학습 보정

즉 단순 차트 뷰어가 아니라 `포트폴리오 + 추천 + 추적 + 학습형 보정` 구조를 가진 앱입니다.

## 2. 실행 방법

### 가장 쉬운 방법

```powershell
& "C:\Users\njwjs\AppData\Local\Programs\Python\Python313\python.exe" -m streamlit run app.py
```

### 가상환경 사용 방법

```powershell
python -m venv .venv
& ".\.venv\Scripts\Activate.ps1"
pip install -r requirements.txt
streamlit run app.py
```

실행 후 기본 주소:

- `http://localhost:8501`

종료:

- 터미널에서 `Ctrl + C`

## 4. 현재 상태 한눈에 보기

현재 버전에서 특히 바로 체감되는 부분은 아래입니다.

- 보유 종목 진단에 현재가와 당일등락이 같이 보입니다.
- 오늘 추천, 실시간, 단타 화면에도 현재가와 시세기준이 같이 보입니다.
- 관심 추적 탭은 현재가, 현재수익률, 목표/손절 상태 요약 카드가 붙어 있습니다.
- 전체 새로고침 버튼으로 가격/추적/학습 캐시를 같이 다시 불러올 수 있습니다.
- 실시간 탭은 분봉 기준 준실시간 스캐너입니다.

즉 지금 단계는 `차트 + 추천 + 추적 + 준실시간 확인`까지 한 번에 보는 개인용 보조 툴에 가깝습니다.
## 5. 사이드바 설명

사이드바에서는 전체 기준을 조정합니다.

### 조회 설정

- 시장: `US`, `KR`
- 종목 코드 입력

예시:

- 미국: `AAPL`, `NVDA`, `VOO`
- 한국: `005930.KS`, `000660.KS`

### 추천 기준

- 최소 추천 점수
- 상위 표시 개수

### 실시간 기준

- 실시간 최소 점수
- 분봉 간격 `1m`, `5m`, `15m`

### 자동 분석 엔진

서버가 켜져 있는 동안 자동 후보군과 일부 장중 데이터를 저장합니다.

표시 항목:

- 실행 상태
- 주기
- 최근 실행 시각
- 최근 결과

버튼:

- `지금 자동 분석 실행`
- `전체 새로고침`

### 전체 새로고침

상단의 `전체 새로고침` 버튼을 누르면 아래 캐시를 같이 비웁니다.

- 일봉 데이터
- 분봉 데이터
- 학습 보정
- 추적 통계
- 현재가 캐시

즉 일부 수치만 바꾸는 게 아니라, 현재 화면 계산을 전체적으로 다시 불러오는 용도입니다.

## 13. 실시간 탭

실시간 탭은 `분봉 기반 준실시간 스캐너`입니다.

중요한 점:

- 초단위 체결형은 아닙니다.
- `1m`, `5m`, `15m` 분봉 기준으로 후보를 압축합니다.
- 장이 닫혀 있거나 데이터가 비어 있으면 후보가 없을 수 있습니다.
- 이 경우에도 오류가 아니라 빈 결과 또는 안내 문구로 보이는 것이 정상입니다.

읽는 법:

- `현재가`: 마지막 분봉 또는 최신 조회 시세
- `당일등락`: 전일 종가 대비 움직임
- `시세기준`: 현재 표에 표시된 가격의 기준 시각
- `실시간 스캐너 새로고침`: 분봉 캐시를 비우고 다시 조회

자동 추천과 별개로, 사용자가 직접 골라 저장한 종목만 따로 추적하는 화면입니다.

### 왜 필요한가

- 자동 추적은 시스템 전체 성과용
- 관심 추적은 내가 실제로 보고 싶은 종목 관리용

### 기능

- 추천 후보에서 직접 추가
- 각 추천/실시간/단타 화면에서 바로 추가
- 메모 저장
- 추가 시점 기록
- 이후 3일 / 5일 / 20일 성과 추적
- 5일 경로 / 20일 경로 확인
- 현재가 / 당일등락 / 현재수익률 확인
- 손절근접 / 목표도달 같은 상태 확인

예:

- 목표가 먼저 갔는지
- 손절이 먼저 났는지
- 시간이 지나며 계속 좋았는지

## 19. 현재 들어간 고급 지표

### 상대강도

최근 20일, 60일, 120일, 1년 흐름을 합성해서 종목의 강한 정도를 봅니다.

용도:

- 손절가 계산
- 단타 리스크 판단
- 종목 흔들림 크기 파악

현재 가격이 1년 고점 대비 얼마나 가까운지 봅니다.

용도:

- 주도주 성격인지

## 23. 문제 생길 때

### 앱이 안 켜질 때

```powershell
& "C:\Users\njwjs\AppData\Local\Programs\Python\Python313\python.exe" -m streamlit run app.py
```

### 멈춘 것처럼 보일 때

Streamlit은 서버가 켜진 뒤 계속 실행 상태로 기다리는 구조입니다.

### 종목이 안 나올 때

- 티커 형식 확인
- 한국 종목은 `.KS`, `.KQ` 확인
- 데이터 제공처에서 당일 데이터가 비어 있을 수 있음
