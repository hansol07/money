# Stock Decision Helper 기능 정리

문서 기준일: 2026-05-16  
목적: 집/회사 PC에서 같은 기준으로 앱을 켜고, 어떤 화면이 어떤 판단을 하는지 빠르게 확인하기 위한 기능 문서입니다.

## 1. 앱의 목표

이 앱은 단순 종목 검색기가 아니라 개인용 투자 의사결정 보조 도구입니다.

- 한국/미국 시장을 나눠서 후보를 봅니다.
- 추천마다 `현재가`, `진입가`, `손절가`, `1~3차 목표가`, `신뢰도`, `가격 기준시각`을 확인합니다.
- 추천 결과를 저장해서 이후 수익률, 목표가 도달, 손절가 이탈 여부를 추적합니다.
- 로컬 DB에 가격 데이터와 추천 스냅샷을 쌓아 다음 추천에 학습 보정으로 반영합니다.

## 2. 메인 화면 구조

현재 큰 화면은 5개 흐름으로 정리되어 있습니다.

| 화면 | 용도 |
|---|---|
| 대시보드 | 오늘 먼저 볼 내용, 데이터 수집, DB 상태, 강한 후보 요약 |
| 신규 추천 | 오늘 추천, 단타, 장기 보유, 배당주, 예산 플랜 |
| 보유 운용 | 보유 종목 입력, 포트폴리오, 물타기/불타기, 보유 기준 예산 |
| 종목 분석 | 단일 종목 차트와 매매 판단 |
| 학습·관리 | 학습 추적, 관심 종목, 뉴스/이벤트, 백테스트, 후보군 관리 |

## 3. 신규 추천

신규 추천은 모두 시장을 `미국`과 `한국`으로 나눠서 보는 방향입니다.

| 추천 방향 | 목적 | 핵심 확인값 |
|---|---|---|
| 오늘 추천 | 오늘 진입 검토 후보 | 현재가, 진입가, 손절가, 목표가, 점수, 신뢰도 |
| 단타 | 짧은 구간 매매 후보 | 분봉 기준, 당일 상승 보정, 추격금지/눌림대기, 손절가 |
| 장기 보유 | 오래 들고 갈 후보 | 추세, 상대강도, 장세, 진입가, 손절가, 목표가 |
| 배당주 | 배당/배당성장 후보 | 배당수익률, 배당성장, 배당락일, 모으기 구간 |
| 예산 플랜 | 입력 예산 기준 매수 계획 | 예산 안의 수량, 배정금액, 진입가, 손절가, 목표가 |

## 4. 보유 운용

보유 운용은 이미 들고 있는 종목 기준으로 판단합니다.

- 보유 입력: 시장, 티커, 종목명, 수량, 평균단가, 추가 예산, 목표비중 저장
- 포트폴리오: 현재가 갱신 후 수익률, 비중, 위험도 확인
- 물타기/불타기: 보유 종목 기준 추가매수, 보류, 축소 판단
- 보유 기준 예산: 기존 보유와 신규 추천을 함께 참고해 추가 예산 배분

## 5. 학습·관리

앱의 학습은 추천 결과를 저장하고 시간이 지난 뒤 실제 성과를 평가하는 방식입니다.

- 추천 결과 저장: 오늘 추천, 예산 플랜, 실시간, 단타, 고위험 단타
- 추적 평가: 1일, 3일, 5일, 20일 수익률과 목표/손절 선도달 여부
- 학습 보정: 잘 맞았던 패턴은 가산, 약했던 패턴은 감점
- 관심 종목: 직접 고른 종목의 현재가, 수익률, 목표/손절 상태 추적
- 백테스트: 특정 종목의 과거 점수 기반 매매 결과 확인

## 6. 데이터 구조

| 저장 위치 | 역할 |
|---|---|
| `data/portfolio.json` | 내 보유 종목 |

| `data/universe_KR.csv` | 한국 후보 유니버스 |

## 7. 로컬 DB 운영 방식

기본 원칙은 `빠르게 보여주기 위해 로컬 DB를 우선 사용하고, 부족한 데이터만 온라인으로 보강`하는 방식입니다.

- 사이드바 `로컬 DB 우선 빠른 조회`: 켜두면 저장된 가격 DB를 먼저 사용합니다.
- 대시보드 `오늘 데이터 한방 수집`: 하루에 한 번 눌러 주요 추천과 스냅샷을 저장합니다.
- 전체 가격 백필: 시간이 오래 걸리므로 서버를 오래 켜둘 수 있을 때 실행합니다.
- 회사 PC 이전: `data/cache.sqlite3`는 Git LFS로 같이 받아야 같은 학습 상태를 이어갈 수 있습니다.

## 8. 실행 명령

일반 실행:

```powershell
python -m streamlit run app.py --server.port=8501 --browser.gatherUsageStats=false
```

스크립트 실행:

```powershell
.\scripts\start_app.ps1
```

회사 PC 최초 세팅:

```powershell
git clone https://github.com/hansol07/money.git
cd money
git lfs install
git lfs pull
python -m pip install -r requirements.txt
.\scripts\start_app.ps1
```

## 9. 현재 주의점

- 이 앱은 투자 판단 보조 도구이며 최종 주문 전에는 증권사 호가와 공시를 확인해야 합니다.
- 실시간 탭은 초단위 체결 시스템이 아니라 분봉 기반 준실시간 스캐너입니다.
- 로컬 DB가 오래되면 추천 신뢰도는 떨어집니다. 대시보드에서 데이터 기준시각을 먼저 확인해야 합니다.
- 집/회사 PC에서 동시에 DB를 수정하고 각각 push하면 SQLite DB 충돌이 날 수 있습니다.
